# InfoSec Awareness Demo (資安意識宣導網頁範例)

這是一個現代化、具備高度科技感（Cyberpunk 風格）的**單一檔案靜態網頁 (Single File HTML)**。

該專案旨在作為企業內部或組織進行「資訊安全意識宣導」的入口網站範本。它具備響應式設計 (RWD)，無需後端資料庫即可運作，便於快速部署與展示。

## ✨ 特色 (Features)

*   **極致科技視覺**：採用 Neon Cyan (青色) 與 Purple (紫色) 的賽博龐克配色，搭配深色模式。
*   **動態 3D 背景**：使用純 CSS 繪製的 3D 透視網格 (Perspective Grid) 與漂浮粒子動畫，解決背景空洞感。
*   **絲滑滾動體驗**：優化的 `scroll-behavior: smooth` 與導覽列定位修正。
*   **玻璃擬態 (Glassmorphism)**：卡片與儀表板採用磨砂玻璃質感，提升視覺層次。
*   **無需依賴框架**：完全使用原生 HTML5 與 CSS3 撰寫，無須安裝 Node.js、Bootstrap 或 Tailwind。
*   **教育互動性**：包含模擬釣魚信件的互動區塊，幫助使用者識別資安威脅。

## 🛠️ 技術堆疊 (Tech Stack)

*   **核心**：HTML5, CSS3 (Flexbox/Grid/Animations)
*   **圖示庫**：FontAwesome 6.4.0 (CDN)
*   **字體**：Google Fonts (Inter, JetBrains Mono)
*   **相依性**：無 (Zero Dependencies)

## 🚀 快速開始 (Quick Start)

由於本專案是單一 HTML 檔案，無需任何伺服器設定。

1.  下載本專案中的 `index.html` 檔案。
2.  雙擊檔案，使用任何現代瀏覽器 (Chrome, Edge, Firefox, Safari) 開啟。
3.  即可瀏覽完整效果。
